본 약관은 molldo(이하 "운영자")가 제공하는 Tiny Speech 서비스 이용과 관련한 권리, 의무 및 책임사항을 정합니다.

1. 서비스 내용

Tiny Speech는 한국어 대화를 쉬운 영어 표현으로 변환하고 학습을 돕는 모바일 앱 서비스입니다.

2. API 키 이용 조건

  • 이용자는 본인이 발급한 Gemini API 키를 입력하여 서비스를 사용할 수 있습니다.
  • 입력된 API 키는 이용자 디바이스에만 저장되며, 운영자는 이를 수집/저장/관리하지 않습니다.
  • 운영자는 이용자의 API 키를 확인할 수 없고 복구할 수 없습니다.

3. 이용자 책임

  • API 키 오입력, 노출, 사용량 초과, 과금 이슈는 이용자와 API 제공자 정책을 따릅니다.
  • 이용자는 API 키를 안전하게 관리할 책임이 있습니다.

4. 데이터 저장

학습 데이터는 이용자 디바이스 로컬 저장소에 저장되며, 서버 DB 동기화 기능은 기본 제공하지 않습니다.

5. 광고

앱에는 Google AdMob 광고가 포함될 수 있으며 관련 처리는 Google 정책이 적용됩니다.

6. 면책

  • 운영자는 외부 API 장애, 정책 변경, 네트워크 상태로 인한 제한에 대해 책임을 지지 않습니다.
  • 운영자는 이용자의 API 키 관리 부주의로 인한 손해에 대해 책임을 지지 않습니다.
