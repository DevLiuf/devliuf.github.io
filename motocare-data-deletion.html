MotoCare 데이터 삭제 요청

MotoCare는 사용자의 데이터를 소중히 여기며, 언제든지 데이터 삭제를 요청할 수 있습니다.

앱 내 데이터 직접 삭제

MotoCare의 대부분의 데이터는 사용자의 기기에만 로컬 저장됩니다. 다음 방법으로 직접 삭제할 수 있습니다.

방법 1: 앱 삭제

1기기에서 MotoCare 앱을 삭제하면 모든 로컬 데이터가 즉시 삭제됩니다.

방법 2: 바이크 데이터 개별 삭제

1앱 → 바이크 관리 → 삭제할 바이크 선택

2"바이크 삭제" 버튼을 누르면 해당 바이크의 모든 주유 기록, 정비 기록이 삭제됩니다.

방법 3: Google Drive 백업 데이터 삭제

1앱 → 설정 → Google 계정 연결 해제

2Google Drive에서 앱 데이터 삭제: Google Drive 설정 → "앱 관리" → MotoCare → "앱 연결 해제"에서 "이 앱에서 사용한 숨겨진 앱 데이터도 삭제" 체크

삭제되는 데이터

데이터 유형 저장 위치 삭제 방법
바이크 정보 (닉네임, 브랜드, 모델 등) 기기 로컬 앱 삭제 또는 바이크 삭제
주유 기록 기기 로컬 앱 삭제 또는 바이크 삭제
정비 기록 기기 로컬 앱 삭제 또는 바이크 삭제
앱 설정 기기 로컬 앱 삭제
Google Drive 백업 Google Drive AppData Google Drive 설정에서 앱 연결 해제
구독 정보 Apple / Google 서버 스토어에서 구독 해지
광고 식별자 광고 네트워크 기기 설정에서 광고 ID 재설정

이메일로 데이터 삭제 요청

위 방법으로 직접 삭제할 수 없거나, 운영자가 보유한 데이터의 완전한 삭제를 원하시는 경우 이메일로 요청해 주세요.

요청 시 포함할 정보:

  • 앱에서 사용한 Google 계정 이메일 (Google Drive 백업을 사용한 경우)
  • 삭제를 원하는 데이터 범위 (전체 또는 특정 데이터)
📧 데이터 삭제 요청 이메일 보내기

주의: 데이터 삭제는 되돌릴 수 없습니다. 삭제 전 필요한 데이터를 백업해 주세요.

처리 기간

이메일로 접수된 삭제 요청은 영업일 기준 7일 이내에 처리되며, 완료 시 이메일로 안내드립니다.
