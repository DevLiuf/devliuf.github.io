AI 연인 케어 비서

시행일: 2026년 2월 22일

개인정보처리방침

§1. 수집하는 개인정보 항목

  • 필수 정보: 이메일, 소셜 로그인 ID (Google), 기기 식별자 (FCM 토큰)
  • 선택 정보: 연인 프로필 (이름, 생일, MBTI, 혈액형, 선호도, 사이즈), AI 편지 대화 내역
  • 민감정보 (별도 동의): 생리주기 데이터 (AES-256 암호화 저장)
  • 자동수집 정보: 앱 사용 통계 (Firebase Analytics), 오류 로그 (Firebase Crashlytics), 광고 식별자 (ATT 동의 시)

§2. 수집 목적

  • AI 맞춤 편지 작성 (Claude API)
  • 기념일/생리주기 로컬 알림 및 FCM 푸시 알림
  • 서비스 개선 및 사용 통계 분석
  • 맞춤형 광고 표시 (무료 사용자)
  • 오류 감지 및 서비스 안정성 확보

§3. 보관 기간 및 파기

  • 회원 탈퇴 시 즉시 삭제
  • AI 편지 대화 내역: 회원 탈퇴 시 삭제
  • 관련 법령에 따른 보관 (전자상거래법 등)
  • Crashlytics 오류 로그: 90일 후 자동 삭제

§4. 제3자 제공

  • Firebase (Google): 인증, 데이터 저장, 푸시 알림, 사용 통계, 오류 추적
  • Claude AI (Anthropic): AI 편지 생성 (Cloud Functions 프록시 경유, API 키 미노출)
  • AdMob (Google): 광고 표시 (프리미엄 사용자 제외)
  • RevenueCat: 구독 결제 관리 (구매 영수증 검증, 구독 상태 동기화)
  • Coupang Partners (쿠팡): 선물샵 제휴 링크 제공 (구매 시 수수료 수익 발생)
  • App Tracking Transparency: iOS 사용자 동의 시 광고 식별자 수집

§5. 이용자 권리

  • 개인정보 열람, 수정, 삭제 요청
  • 마케팅 동의 철회
  • 계정 삭제 (설정 > 계정 삭제)

§6. 민감정보 처리

  • 생리주기 데이터는 AES-256 암호화
  • 기기 내 Secure Storage에 암호화 키 보관
  • 별도 동의 없이 수집하지 않음

§7. 앱 추적 투명성 (ATT)

  • iOS 14.5 이상 기기에서 앱 추적 권한 요청
  • 거부 시에도 서비스 이용에 제한 없음
  • 동의 시 맞춤형 광고 제공에 활용
