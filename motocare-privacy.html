MotoCare 개인정보처리방침

시행일자: 2024년 1월 1일

최종 수정일: 2025년 11월 25일

1. 개인정보의 처리 목적

MotoCare("이하 "앱")은 다음의 목적을 위하여 개인정보를 처리합니다. 처리하고 있는 개인정보는 다음의 목적 이외의 용도로는 이용되지 않으며, 이용 목적이 변경되는 경우에는 별도의 동의를 받는 등 필요한 조치를 이행할 예정입니다.

1.1 오토바이 정비 관리 서비스 제공

  • 사용자의 오토바이 정비 기록 저장 및 관리
  • 주유 기록 및 연비 계산
  • 정비 알림 서비스 제공
  • 통계 및 보고서 생성

1.2 구글 드라이브 백업 서비스 (선택적)

  • 사용자가 선택한 경우에 한하여 데이터를 구글 드라이브에 백업
  • 다른 기기에서 데이터 복원

2. 수집하는 개인정보 항목

2.1 로컬 저장 데이터 (기본 기능)

MotoCare는 기본적으로 모든 데이터를 사용자 기기에만 저장합니다. 서버로 전송되지 않으며, 외부에 공유되지 않습니다.

항목 수집 목적
오토바이 정보 (닉네임, 제조사, 모델, 연식 등) 정비 기록 관리
주행거리 기록 정비 주기 계산 및 알림
주유 기록 (날짜, 주행거리, 주유량, 비용, 유종) 연비 계산 및 통계
정비 기록 (항목, 날짜, 주행거리, 메모) 정비 이력 관리
사용자 설정 (단위 시스템, 알림 설정) 앱 사용 환경 개인화

2.2 구글 계정 정보 (선택적 - 백업 기능 사용 시)

사용자가 구글 드라이브 백업 기능을 선택한 경우에만 수집됩니다:

  • 구글 계정 이메일: 계정 식별 및 로그인 상태 표시
  • 구글 드라이브 AppData 폴더 접근 권한: 백업 파일 저장 (앱 전용 폴더, 다른 파일 접근 불가)

중요: 구글 로그인 시 수집되는 정보는 구글 OAuth 2.0 표준을 따르며, MotoCare 서버로 전송되지 않습니다. 백업 데이터는 구글 드라이브의 AppData 폴더에만 저장되며, 사용자만 접근할 수 있습니다.

3. 개인정보의 처리 및 보유 기간

3.1 로컬 데이터

  • 보유 기간: 사용자가 앱을 삭제할 때까지
  • 삭제 방법: 앱 삭제 시 모든 로컬 데이터 자동 삭제

3.2 구글 드라이브 백업 데이터

  • 보유 기간: 사용자가 직접 삭제할 때까지
  • 삭제 방법:
    • 앱 내 "구글 드라이브 백업 삭제" 기능 사용
    • 구글 계정 로그아웃 후 구글 드라이브에서 수동 삭제
    • 구글 계정 자체 삭제 시 자동 삭제

4. 개인정보의 제3자 제공

MotoCare는 사용자의 개인정보를 제3자에게 제공하지 않습니다.

단, 다음의 경우는 예외로 합니다:

  • 사용자가 사전에 동의한 경우
  • 법령의 규정에 의거하거나, 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우

5. 개인정보의 처리 위탁

MotoCare는 원활한 서비스 제공을 위해 다음과 같이 개인정보 처리업무를 위탁하고 있습니다:

수탁업체 위탁업무 내용 개인정보 보유 및 이용 기간
Google LLC 구글 드라이브 백업 저장 (선택적) 사용자가 백업 삭제 시까지
Google LLC Google Ads 광고 서비스 (선택적, 무료 버전) 광고 표시 시점까지
Google LLC (Play Console) 인앱 구매 처리 (Android) 구매 시점부터 환불 기간 종료까지
Apple Inc. (App Store) 인앱 구매 처리 (iOS) 구매 시점부터 환불 기간 종료까지

6. 정보주체의 권리·의무 및 행사방법

사용자는 개인정보주체로서 다음과 같은 권리를 행사할 수 있습니다:

6.1 개인정보 열람 요구

  • 앱 내 "통계" 및 "바이크 관리" 메뉴에서 저장된 모든 데이터 확인 가능

6.2 개인정보 정정·삭제 요구

  • 앱 내에서 직접 수정 및 삭제 가능
  • 바이크 삭제 시 해당 바이크의 모든 데이터 삭제

6.3 개인정보 처리정지 요구

  • 구글 드라이브 백업 기능 해제 (앱 내 설정)
  • 알림 기능 비활성화 (기기 설정)

6.4 개인정보 이동 요구

  • 앱 내 "보고서 생성" 기능을 통해 PDF 내보내기 가능

7. 개인정보의 파기

7.1 파기 절차

사용자가 앱을 삭제하면 모든 로컬 데이터가 자동으로 파기됩니다.

7.2 파기 방법

  • 전자적 파일: 기기 저장소에서 완전 삭제
  • 구글 드라이브 백업: 사용자가 직접 삭제하거나, 앱 내 "백업 삭제" 기능 사용

8. 개인정보 보호책임자

MotoCare는 개인정보 처리에 관한 업무를 총괄해서 책임지고, 개인정보 처리와 관련한 정보주체의 불만처리 및 피해구제 등을 위하여 아래와 같이 개인정보 보호책임자를 지정하고 있습니다.

개인정보 보호책임자

  • 이름: MotoCare 개발팀
  • 연락처: [이메일 주소를 여기에 입력하세요]

정보주체는 MotoCare 이용 중 발생한 모든 개인정보 보호 관련 문의, 불만처리, 피해구제 등에 관한 사항을 개인정보 보호책임자에게 문의하실 수 있습니다.

9. 개인정보의 안전성 확보조치

MotoCare는 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다:

9.1 기술적 조치

  • 로컬 저장소 암호화: 기기 내 데이터는 운영체제의 보안 기능으로 보호
  • 구글 드라이브 암호화: 백업 데이터는 Google의 표준 암호화 적용
  • HTTPS 통신: 구글 API 통신 시 암호화된 연결 사용

9.2 관리적 조치

  • 개인정보 접근 권한 최소화 (사용자 본인만 접근 가능)
  • 정기적인 보안 업데이트

10. 개인정보 자동 수집 장치의 설치·운영 및 거부

10.1 Google Ads (무료 버전)

무료 버전 사용자에게는 맞춤형 광고를 제공하기 위해 다음 정보가 수집될 수 있습니다:

  • 광고 식별자 (IDFA/AAID): 익명화된 광고 타게팅 용도
  • 거부 방법:
    • iOS: 설정 → 개인정보 보호 → 추적 → "앱이 추적을 요청하도록 허용" 비활성화
    • Android: 설정 → Google → 광고 → "광고 맞춤설정 선택 해제"
    • 또는 유료 버전 구매 시 모든 광고 제거

11. 아동의 개인정보 보호

MotoCare는 만 14세 미만 아동의 개인정보를 수집하지 않습니다. 오토바이는 면허가 필요한 교통수단이므로, 본 앱은 만 18세 이상 성인을 대상으로 합니다.
