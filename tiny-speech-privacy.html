개인정보 처리방침

시행일: 2026-04-16

Tiny Speech(이하 "앱")은 이용자의 개인정보를 중요하게 생각하며, 관련 법령을 준수합니다.

1. 개인정보처리자 정보

  • 개발사: molldo
  • 문의 이메일: molldo2025@gmail.com

2. 수집 및 처리 정보

  • 이용자가 앱에 직접 입력한 Gemini API 키
  • 학습 데이터(대화 내용, 변환 결과, 핵심 표현)
  • 앱 동작에 필요한 설정값

3. API 키 처리 방식

  • Gemini API 키는 이용자 디바이스 로컬 저장소(SharedPreferences)에만 저장됩니다.
  • 해당 API 키는 앱 운영 서버로 전송되거나 별도 수집/보관되지 않습니다.
  • 외부 API 호출은 이용자 디바이스에서 Google API로 직접 이루어집니다.
  • 이용자는 앱 설정에서 API 키를 언제든지 변경 또는 삭제할 수 있습니다.

4. 학습 데이터 저장

  • 학습 데이터는 이용자 디바이스 로컬 저장소(Hive)에 저장됩니다.
  • 앱은 별도 서버 DB를 통한 학습 데이터 저장을 기본 제공하지 않습니다.

5. 제3자 제공 및 처리위탁

앱은 API 키와 학습 데이터를 앱 서버에서 수집하지 않으며, 해당 데이터에 대한 별도 판매/대여/제공을 하지 않습니다.

단, 이용자 디바이스에서 Google API를 호출하는 과정에는 Google의 정책이 적용됩니다.

6. 광고

앱은 Google AdMob 광고를 포함할 수 있으며, 광고 SDK 처리에는 Google 정책이 적용됩니다.

7. 이용자 권리

  • API 키 변경 또는 삭제
  • 학습 데이터 삭제
  • 앱 삭제를 통한 로컬 데이터 제거

8. 방침 변경

본 방침은 법령 또는 서비스 변경에 따라 수정될 수 있으며, 변경 시 앱 또는 배포 페이지를 통해 고지합니다.
